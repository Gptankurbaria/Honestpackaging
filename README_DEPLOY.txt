
📦 Box Costing Factory Deployment Guide
=======================================

1. SYSTEM REQUIREMENTS
----------------------
- Windows 10 or 11
- No Python installation required (bundled)
- Minimum 4GB RAM recommended

2. HOW TO RUN
-------------
- Open the "BoxCosting" folder.
- Double-click on "BoxCosting.exe".
- A console window will open briefly to start the server.
- The dashboard will be available at http://localhost:8501 in your browser.

3. DATA SAFETY & BACKUPS
-------------------------
- Go to "User Details" -> "Database Backup & Restore".
- Set a custom "Local Storage Path" (e.g., D:/Backups).
- Click "Backup Now" regularly before major updates.
- The system automatically checks for a daily backup on startup.

4. UPDATING RATES
-----------------
- Use the "Masters" menu to update Paper Rates and Operation Costs.
- Changes reflect immediately in new quotations.

5. TROUBLESHOOTING
------------------
- If the app doesn't open, check the "error.log" file in the main folder.
- Ensure no other application is using Port 8501.
- To reset everything, you can delete "box_costing.db" (Caution: This will erase all data).
